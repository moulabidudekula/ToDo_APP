# ✅ ToDo App

A minimalist and responsive ToDo application built with **vanilla JavaScript**, **HTML**, and **CSS**. It supports full **CRUD operations** and saves tasks in **Local Storage**.

## 🔗 Live Demo

[View Live](https://github.com/moulabidudekula/ToDo_APP.git)

---

## 🚀 Features

- 📝 Add, edit, delete tasks
- ✅ Mark tasks as completed
- 💾 Stores tasks using Local Storage
- 📱 Mobile-friendly responsive UI
- ⚡ Lightning-fast performance (no framework)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API
